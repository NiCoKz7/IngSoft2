# 📘 Ingeniería del Software II

Repositorio correspondiente a la materia **Ingeniería del Software II** de 2° año de la **Tecnicatura Superior en Desarrollo de Software** del Instituto Superior de Profesorado N°20.

## 🎯 Objetivo

El objetivo de este repositorio es centralizar los trabajos prácticos, ejercicios y proyectos desarrollados durante la cursada, aplicando conceptos de ingeniería de software enfocados en el desarrollo web.

## 🛠️ Tecnologías utilizadas

Durante la materia se trabajará principalmente con:

- **HTML5** → Estructura de páginas web  
- **CSS3** → Estilos y diseño visual  
- **JavaScript** → Lógica e interactividad  
- **Bootstrap** → Framework para diseño responsive
